# 种花小游戏（PhaserJS）

一个基于 `PhaserJS` 的轻量种花养成小游戏。

你可以在花盆中种下不同花种，等待成长后收取花朵、获取经验升级、解锁更多花盆，并通过花朵兑换道具（浇水壶/化肥）来加速全局成长。

## 功能概览

- 多花种种植：玫瑰、雏菊、牡丹
- 花朵 5 阶段成长，成熟后可收取（点击花或花盆都可）
- 收花随机数量：每次收取 `1~3` 朵
- 经验与等级系统：每朵花提供经验，升级解锁新花盆
- 顶部 HUD：等级、经验条、仓库（各花种分行展示）
- 房子按钮弹窗：查看累计收花总数
- 交易系统：
  - `50` 朵花兑换 `1` 把浇水壶
  - `100` 朵花兑换 `1` 包化肥
- 快捷道具按钮（右上角）：
  - 一键施肥：所有已种植花朵直达最终阶段
  - 一键浇水：所有已种植花朵前进到下一阶段

## 本地运行

这是一个纯前端静态项目，直接用本地静态服务器打开即可。

示例（任选其一）：

1. VS Code 安装并使用 `Live Server`
2. 使用任意静态服务器（如 `npx serve`）

启动后访问 `index.html`。

## 核心规则

- 初始解锁花盆数：`4`
- 总花盆数：`20`
- 每级经验：`100`
- 最高等级：`17`
- 每朵花经验：`5`
- 满级后显示“已满级，已解锁全部花盆”

## 存档说明

游戏数据使用 `localStorage` 存储，键名为：

- `flower_grade_save_v1`

主要存档字段包括：

- `level` / `exp`
- `warehouse`（各花种库存）
- `selectedFlowerType`
- `unlockedPotCount`
- `tools`（`wateringCan`、`fertilizer`）
