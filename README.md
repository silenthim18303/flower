# 花期乘风录

基于 `Phaser 3` 的轻量种花养成小游戏。

在花园中播种、浇灌、收获花朵，升级解锁更多花盆，探索丰富的游戏内容。

## 游戏特色

- 四种花种：玫瑰、雏菊、牡丹、金花
- 牡丹有概率突变为稀有金花
- 花农兑换、送给明星、寻花小游戏等丰富玩法
- 背景音乐循环播放
- 新手引导系统
- 花盆状态自动保存

## 核心玩法

### 种花系统

| 花种 | 获取方式 | 特殊说明 |
|------|----------|----------|
| 玫瑰 | 直接播种 | 基础花种 |
| 雏菊 | 直接播种 | 基础花种 |
| 牡丹 | 直接播种 | 有 1% 概率突变为金花 |
| 金花 | 牡丹突变 / 花农兑换 | 不可直接播种 |

**生长机制：**
- 每株花经历 5 个生长阶段
- 每阶段间隔 10~15 秒随机
- 成熟后点击收获，获得 1~3 朵随机数量
- 每次收获获得 5 点经验

### 升级系统

- 初始花盆：4 个
- 花盆总数：20 个（5 行 × 4 列）
- 每升 1 级解锁 1 个花盆
- 每级所需经验：100
- 最高等级：17
- 满级后显示"已满级，已解锁全部花盆"

### 道具系统

| 道具 | 效果 | 获取方式 |
|------|------|----------|
| 浇水壶 | 所有花朵前进一个阶段 | 50 朵花兑换 / 每日任务 |
| 化肥 | 所有花朵直接成熟 | 100 朵花兑换 / 每日任务 |
| 一键播种 | 自动为空花盆播种 | 500 朵花永久解锁 |
| 一键采摘 | 自动收获成熟花朵 | 500 朵花永久解锁 |

### 每日任务

- 每日累计收花 30 朵
- 奖励：化肥 ×1
- 按自然日自动重置

### 花农兑换

- 入口：房子右侧花农 NPC
- 500 朵玫瑰 = 1 朵金花
- 500 朵雏菊 = 1 朵金花
- 牡丹不参与兑换

### 送给明星

- 入口：种子工具栏"送给明星"
- 独立页面，20 位明星可选
- 每次赠送消耗指定数量金花
- 赠送记录永久保存

### 寻花小游戏

- 入口：花仙子 NPC → 寻花
- 扫雷玩法，揭开草皮寻找隐藏金花
- 门票：10 朵花
- 通关奖励：30 朵玫瑰
- 自动标记功能

## NPC 系统

| NPC | 位置 | 功能 |
|-----|------|------|
| 花农 | 房子右侧 | 金花兑换 |
| 花仙子 | 左侧面板 | 帮助、寻花游戏 |
| 设置按钮 | 左下角 | 音乐控制、关于游戏 |

## 操作说明

- **点击花盆**：空花盆播种 / 成熟花朵收获
- **右键/长按**：标记旗帜（寻花游戏）
- **点击房子**：查看收花统计、兑换道具
- **点击花农**：打开金花兑换
- **点击花仙子**：帮助、寻花游戏
- **点击设置**：音乐控制、关于游戏

## 技术说明

- 引擎：Phaser 3.80.1
- 渲染：WebGL / Canvas 自动切换
- 抗锯齿：启用 `antialias`
- 存储：localStorage 本地存档
- 响应式：适配各种屏幕尺寸
