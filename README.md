# 种花小游戏（PhaserJS）

一个基于 `PhaserJS` 的轻量种花养成小游戏。

玩家可在花盆里播种并收花，获得经验升级、解锁花盆；也可以通过交易和快捷功能提升效率。

## 当前功能

- 花种：`玫瑰`、`雏菊`、`牡丹`（可播种）
- 特殊花：`金花`（不可手动播种，只能由牡丹突变获得）
- 每株花 `5` 阶段成长，点击花或花盆可交互
- 生长间隔为随机 `10~15` 秒（每次阶段推进随机）
- 普通收花随机 `1~3` 朵
- 等级系统：收花获得经验，升级逐步解锁花盆
- 房子按钮弹窗：查看总收花并进行交易
- 右上快捷按钮：一键施肥 / 一键浇水 / 一键播种 / 一键采摘
- 每日任务：独立任务卡片 + 任务弹窗领取奖励

## 关键玩法规则

### 升级与花盆

- 初始解锁花盆：`4`
- 花盆总数：`20`
- 每级经验：`100`
- 每朵花经验：`5`
- 最高等级：`17`
- 满级后经验封顶并显示“已满级，已解锁全部花盆”

### 牡丹突变金花

- 配置位于 `js/config.js -> MUTATION`
- 当前概率：`peonyToGoldenChance = 0.05`（即每次牡丹阶段推进有 5% 概率突变）
- 触发时机：牡丹在生长推进过程中（自然成长 / 一键浇水 / 一键施肥）
- 金花收获结算：`10` 朵牡丹 + `1` 朵金花

### 交易系统

在房子弹窗中消耗花朵兑换道具：

- `50` 朵花兑换 `1` 个 `浇水壶`
- `100` 朵花兑换 `1` 个 `化肥`
- `500` 朵花永久解锁 `一键播种`
- `500` 朵花永久解锁 `一键采摘`

当花朵不足时，兑换按钮会禁用。

### 快捷按钮（右上）

- 一键施肥：消耗 `化肥 x1`，所有已种花朵直接成熟
- 一键浇水：消耗 `浇水壶 x1`，所有已种花朵前进一个阶段
- 一键播种：需先永久解锁，对所有空花盆播种当前选中花种
- 一键采摘：需先永久解锁，采摘所有成熟花朵

### 每日任务

- 任务配置：`js/config.js -> DAILY_TASK`
- 当前任务：每日累计收花 `30` 朵
- 当前奖励：`化肥 x1`
- 展示方式：右侧独立“每日任务”卡片（位于快捷按钮区）
- 点击卡片打开任务弹窗，满足条件后可领取
- 任务状态按自然日重置（本地时间），并持久化到本地存档

## 本地运行

这是纯前端静态项目，使用任意静态服务器启动即可：

1. VS Code `Live Server`
2. `npx serve`

然后访问 `index.html`。

## 存档说明

使用 `localStorage`，键名：`flower_grade_save_v1`

主要字段：

- `level` / `exp`
- `warehouse`（含 `rose` / `daisy` / `peony` / `golden`）
- `selectedFlowerType`（仅可播种花种）
- `unlockedPotCount`
- `tools`
  - `wateringCan`（数量）
  - `fertilizer`（数量）
  - `oneClickPlant`（是否永久解锁）
  - `oneClickHarvest`（是否永久解锁）
- `dailyTask`
  - `dateKey`
  - `collectProgress`
  - `rewardClaimed`
