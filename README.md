# 种花小游戏（PhaserJS）

一个基于 `PhaserJS` 的轻量种花养成小游戏。

玩家可在花盆里播种并收花，获得经验升级、解锁花盆；也可以通过交易和快捷功能提升效率。

## 当前功能

- 花种：`玫瑰`、`雏菊`、`牡丹`（可播种）
- 特殊花：`金花`（不可手动播种，只能由牡丹突变获得）
- 每株花 `5` 阶段成长，点击花或花盆可交互
- 生长间隔为随机 `10~15` 秒（每次阶段推进随机）
- 普通收花随机 `1~3` 朵
- 等级系统：收花获得经验，升级逐步解锁花盆
- 房子按钮弹窗：查看总收花并进行交易
- 右上快捷按钮：一键施肥 / 一键浇水 / 一键播种 / 一键采摘（缩小 20%）
- 每日任务：经验卡右侧独立任务卡片 + 任务弹窗领取奖励
- 舞台入口：种子工具栏中的"送给明星"卡片
- 花农入口：房子左侧花农 NPC，点击打开金花兑换弹窗
- 左上信息区：经验卡 + 独立可折叠仓库卡（缩小 20%）
- 启动加载页：首屏显示进度条，单次会话内仅显示一次
- 弹窗动效：收花统计 / 每日任务 / 花农兑换支持打开关闭动画
- 折叠动效：仓库卡片支持平滑展开与收起
- 花盆状态持久化：切换页面或关闭后花盆中的花朵状态自动保存恢复
- 页面离开自动保存：`beforeunload` 和 `visibilitychange` 事件触发时自动存档

## 关键玩法规则

### 升级与花盆

- 初始解锁花盆：`4`
- 花盆总数：`20`
- 每级经验：`100`
- 每朵花经验：`5`
- 最高等级：`17`
- 满级后经验封顶并显示“已满级，已解锁全部花盆”
- 未解锁花盆使用专用贴图：`img/背景/未解锁花盆.png`

### 牡丹突变金花

- 配置位于 `js/config.js -> MUTATION`
- 当前概率：`peonyToGoldenChance = 0.01`（即每次牡丹阶段推进有 1% 概率突变）
- 触发时机：牡丹在生长推进过程中（自然成长 / 一键浇水 / 一键施肥）
- 金花收获结算：`10` 朵牡丹 + `1` 朵金花

### 交易系统

在房子弹窗中消耗花朵兑换道具：

- `50` 朵花兑换 `1` 个 `浇水壶`
- `100` 朵花兑换 `1` 个 `化肥`
- `500` 朵花永久解锁 `一键播种`
- `500` 朵花永久解锁 `一键采摘`

当花朵不足时，兑换按钮会禁用。

### 快捷按钮（右上）

- 一键施肥：消耗 `化肥 x1`，所有已种花朵直接成熟
- 一键浇水：消耗 `浇水壶 x1`，所有已种花朵前进一个阶段
- 一键播种：需先永久解锁，对所有空花盆播种当前选中花种
- 一键采摘：需先永久解锁，采摘所有成熟花朵

### 每日任务

- 任务配置：`js/config.js -> DAILY_TASK`
- 当前任务：每日累计收花 `30` 朵
- 当前奖励：`化肥 x1`
- 展示方式：经验卡右侧独立“每日任务”卡片
- 点击卡片打开任务弹窗，满足条件后可领取
- 任务状态按自然日重置（本地时间），并持久化到本地存档

### 金花兑换

- 入口位置：房子左侧花农 NPC（`img/npc/花农.png`）
- 兑换规则：`500` 朵玫瑰 `或` `500` 朵雏菊可兑换 `1` 朵金花
- 牡丹不参与兑换
- 点击花农打开卡片式兑换弹窗（玫瑰兑换 / 雏菊兑换）
- 弹窗标题为“花农兑换”，具体兑换按钮按库存独立禁用

### 送给明星页面

- 入口位置：种子工具栏中的"送给明星"卡片
- 点击后跳转到独立页面：`donate.html`
- 页面背景使用 `img/背景/加载背景.png`，固定不随滚动
- 页面展示 `img/人物形象/` 下的 `20` 位明星头像与姓名
- 打赏规则：每次打赏消耗 `1` 朵金花
- 金花来源：通过花农兑换获得（玫瑰/雏菊 `500` 兑换 `1` 金花）
- 打赏后实时扣减 `warehouse.golden` 并保存到游戏存档
- 明星打赏次数独立持久化到 `localStorage` 键：`flower_donate_records_v1`
- 返回游戏按钮：保存当前状态后跳转回 `index.html`

### 收花统计（房子弹窗）

- 顶部显示累计收花总数
- 明细展示各花种累计数量：`玫瑰 / 雏菊 / 牡丹 / 金花`

### 仓库卡片

- 仓库从经验卡中独立为单独卡片
- 默认折叠，点击标题可展开/收起

## 本地运行

这是纯前端静态项目，使用任意静态服务器启动即可：

1. VS Code `Live Server`
2. `npx serve`

然后访问 `index.html`。

## 存档说明

使用 `localStorage`，键名：`flower_grade_save_v1`

主要字段：

- `level` / `exp`
- `warehouse`（含 `rose` / `daisy` / `peony` / `golden`）
- `selectedFlowerType`（仅可播种花种）
- `unlockedPotCount`
- `tools`
  - `wateringCan`（数量）
  - `fertilizer`（数量）
  - `oneClickPlant`（是否永久解锁）
  - `oneClickHarvest`（是否永久解锁）
- `dailyTask`
  - `dateKey`
  - `collectProgress`
  - `rewardClaimed`
- `potStages`（花盆阶段数组，`-1` 未解锁 / `0` 空花盆 / `1~5` 生长阶段）
- `potFlowerTypes`（花盆花种数组，`null` 或花种 id）

其他本地存储：

- `flower_donate_records_v1`：记录每位明星累计收到的金花数量
